## 정적 시드 vs CMS

다음 마스터 데이터는 **출시 시 시드 → 운영자 CMS로 이관** 검토:

| 엔티티              | 초기 운영                            |
| ------------------- | ------------------------------------ |
| `Mission`           | 시드 50건 → CMS                      |
| `MentalCareContent` | 시드 30건 → CMS                      |
| `GrowthIndicator`   | 시드 (CDC/AAP 기반 표) → 분기별 갱신 |
| `ImprovementTip`    | 시드 → CMS                           |
| `PeerInsight`       | 시드 → 분기별 통계 갱신              |
| `InspirationQuote`  | 시드                                 |
