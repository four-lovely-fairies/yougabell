# Child — 자녀

> 한 명의 사용자가 다자녀를 가질 수 있다. 미션·로드맵·리포트는 자녀 단위로 구성된다.

---

## Child

> 출처: 온보딩03 (`851:6893`), 홈 (`851:3866` "김유스(7살 여)", `851:4603` "김어스(4살 남)")

| 필드           | 타입                    | 필수 | 설명                                             | 출처/메모                               |
| -------------- | ----------------------- | :--: | ------------------------------------------------ | --------------------------------------- |
| `id`           | `string` (UUID)         |  \*  | PK                                               | —                                       |
| `userId`       | `FK → User.id`          |  \*  | 부모 ID                                          | —                                       |
| `name`         | `string`                |  \*  | 예: "김유스"                                     | `851:6893`                              |
| `birthDate`    | `Date`                  |  \*  | 예: `2024-01-02`. 월령/연령 계산 기준            | `851:6893`                              |
| `gender`       | `enum('female','male')` |  \*  | "여아" / "남아"                                  | `851:6893`                              |
| `notes`        | `string`                |  ?   | 특이사항 (알레르기, 발달 이슈 등). 디폴트 "없음" | `851:6893` "특이사항: 없음"             |
| `displayOrder` | `number`                |  \*  | 다자녀 시 홈 슬라이드 순서                       | `851:5266` "다자녀일 경우 ... 슬라이드" |
| `createdAt`    | `DateTime`              |  \*  | —                                                | —                                       |

### 파생 값 (저장 X, 계산)

| 필드            | 계산                          | 사용처                          |
| --------------- | ----------------------------- | ------------------------------- |
| `ageYears`      | `today - birthDate` (만 나이) | 홈 카드 "(7살 여)" `851:3866`   |
| `ageMonths`     | `today - birthDate` (월령)    | 로드맵 "7개월 차" `851:5028`    |
| `growthStageId` | 월령 → `GrowthStage` 매핑     | 로드맵 "자아 형성기" `851:5028` |

### 관계 (Relations)

- N:1 ← `user: User` _(via `userId`)_
- 1:N → `missionExecutions: MissionExecution[]`
- 1:N → `weeklyReports: WeeklyReport[]`
